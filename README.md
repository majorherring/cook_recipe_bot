# Telegram-бот «Книга рецептов»

## Описание проекта

Telegram-бот «Книга рецептов» — это программное обеспечение, разработанное на языке Python для хранения, поиска и просмотра кулинарных рецептов.

Бот позволяет пользователю добавлять собственные рецепты, указывать категорию блюда, список ингредиентов и способ приготовления. Все данные сохраняются в базе данных SQLite.

Проект выполнен в рамках практико-ориентированной работы индивидуально.

## Функциональные возможности

Бот реализует следующие функции:

- добавление нового рецепта;
- выбор категории рецепта;
- сохранение ингредиентов и способа приготовления;
- просмотр всех сохранённых рецептов;
- просмотр подробной информации о рецепте;
- поиск рецепта по названию;
- фильтрация рецептов по категории;
- удаление рецептов;
- хранение данных в базе SQLite.

## Используемые технологии

| Компонент | Технология |
|---|---|
| Язык программирования | Python |
| Библиотека для Telegram-бота | aiogram 3.x |
| База данных | SQLite |
| Работа с переменными окружения | python-dotenv |
| Тестирование | unittest |
| Контроль версий | Git |
